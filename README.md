# redis-combat
spring boot2.0+shiro+redis 前后端分离项目，Shiro-Redis实现分布式Session共享

本文使用框架如下：
*核心框架：spring boot 2.0.0, spring

mvc框架：spring mvc*
*持久层框架：mybatis
    数据库连接池：alibaba druid
    安全框架：apache shiro
    缓存框架：redis
    日志框架：logback
